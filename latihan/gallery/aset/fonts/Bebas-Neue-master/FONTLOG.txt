FONTLOG for Bebas Neue font.

This file provides detailed information on the Bebas Neue font.
This information should be distributed along with the Bebas Neue font(s) and any
derivative works.


Basic Font Information

Bebas Neue is a free display font for headline, caption, and titling designed by Ryoichi Tsunekawa.
To contribute to the project, Please contact to Dharma Type at https://github.com/dharmatype/Bebas-Neue/ or visit http://dharmatype.com
